# Auto-Bunny-Hop-L4D2
Auto Bunny Hop interno desarrollado en C++ para Left 4 Dead 2. Lee m_fFlags desde client.dll para detectar cuando el jugador toca el suelo y ejecutar el salto en el frame exacto, logrando un bhop rapido y estable.  CTRL - bhop  INSERT on/ff - F11 salir
